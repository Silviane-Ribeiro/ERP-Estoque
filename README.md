#Sistema de Controle de Estoque – Python
Este projeto é um sistema de gerenciamento de estoque via terminal, desenvolvido em Python
Permite cadastrar itens, controlar movimentações, visualizar dados, gerar alertas e exportar os resultados para Excel e CSV.

#Funcionalidades
 Código do produto
 Descrição
 Categoria
 Unidade
 Quantidade inicial
 Preço unitário

#Movimentações de Estoque 
(Entrada, Saída, Bloqueia saída acima do estoque baixo)

#Relatórios Gerenciais
(Giro de estoque, Custo de manutenção, Tempo médio de reposição, Estoque de segurança)

#Visualização
(Pilha, Lista organizada, Alerta automático para estoque baixo)

#Exportação
(Excel, CSV)

#Tecnologias utilizadas
(Python, Pandas, Matplotlib)
